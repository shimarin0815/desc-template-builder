概要欄テンプレートビルダー

🚀 アプリの概要
「概要欄テンプレートビルダー」は、YouTubeや各種SNSで投稿する際に毎回必要となる情報を、ドラッグ＆ドロップで簡単に作成できるツールです。
いつも使う「SNSリンク」「機材リスト」「定型挨拶」などをパーツとして登録し、それらを組み合わせて、あっという間にオリジナルの概要欄のひな形を作ることができます。
もう毎回同じ文章をコピー＆ペーストしたり、手打ちしたりする必要はありません！
✨ 主な機能
パーツ登録: よく使うテキスト（挨拶、リンク、機材など）を自由に登録・編集できます。
ドラッグ＆ドロップ: 登録したパーツを直感的に並べ替えて、概要欄を作成できます。
リアルタイムプレビュー: 作成中の概要欄の見た目をすぐに確認できます。
ワンクリックコピー: 完成した概要欄のテキストをボタン一つでコピーできます。
⚙️ 技術スタック
このアプリケーションは、以下の技術を使って作られたシンプルなシングルページアプリケーションです。
HTML: アプリの骨組み
CSS: アプリのデザインと見た目
JavaScript: アプリの動きと機能
複雑なフレームワークは使用せず、基本的なウェブ技術のみで構築されているため、動作が軽く、理解しやすい構造になっています。
💡 使い方
index.html ファイルをウェブブラウザで開きます。
左側の「登録済みパーツ」から、使いたいパーツを右側の「今日の概要欄」エリアにドラッグ＆ドロップします。
パーツを並べ替えて、自分だけの概要欄のひな形を作成します。
右下の「プレビュー」で内容を確認し、「コピー」ボタンをクリックして、各SNSやYouTubeの概要欄に貼り付けてください。
新しいパーツを追加する
左下にある「<i class="fas fa-plus"></i> 新しいパーツを登録」ボタンをクリックすると、新しいパーツの名前と内容を登録できます。
